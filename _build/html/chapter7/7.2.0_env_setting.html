개발환경#

계좌개설#

안드로이드 혹은 앱 스토어에서 Fig 1. 에서 보이는 한국투자증권 애플리케이션을 설치 합니다. 앱 설치 완료 후, 비대면 계좌 개설을 진행 합니다. 신분증 촬영과 간단한 인증절차를 밟으면 어렵지 않게 계좌 개설을 완료 할 수 있습니다.

Open API 인증키 발급#

이어서 한국투자증권의 Open API 키를 받아야 합니다. 국내 유일 ‘REST API, Websocket’ 방식이어서 기존 증권사 API 처럼 프로그램 설치가 필요 없습니다. Web 을 통해서 데이터를 송/수신 하기 때문에 OS(윈도우, 맥, 리눅스) 에 상관 없이 사용 가능합니다. KIS Developers 홈페이지 (http://apiportal.koreainvestment.com)로 이동한 후, API 신청을 진행 합니다.

[API신청] 클릭 후, Fig 3 에서 보이는 스마트폰 인증을 통해 간단히 로그인을 할 수 있습니다.

로그인을 성공하면 모바일 [인증번호 요청] 을 클릭하고 인증번호 입력 후, [신청] 버튼을 클릭합니다.

[신청] 클릭 후, 개인정보 및 고객 이용약관에 모두 동의한 후, [다음] 버튼을 클릭합니다.

[다음] 클릭 후, 고객명, 이메일, 핸드폰번호, 전화번호 정보가 제대로 기입되었는지 확인하고, 계좌정보란에서 이용할 종합게좌를 선택하고 비밀번호를 입력 합니다. 모의계좌도 체크하여 임의의 계좌번호를 입력 합니다. API그룹란에서 모든 그룹을 선택하고 [다음] 번튼을 클릭합니다.

[다음] 클릭 후, 스마트폰 인증을 완료하면 카톡을 통해 API 신청 완료 메시지를 받을 수 있습니다. 이로써 한국투자증권의 Open API 신청을 완료했습니다.

디스코드 설정#

자동매매 경과를 모니터링 할 수 있는 디스코드 설정을 진행 하겠습니다. 먼저 디스코드 홈페이지(https://discord.com/) 를 방문하여 회원가입을 진행 합니다. 회원가입을 완료 후, [웹브라우저에서 Discord 열기] 를 클릭합니다. 디스코드 웹 APP 및 모바일 APP 을 다운 받으면 메시지를 더 편리하게 받아 볼 수 있습니다.

웹브라우저에서 디스코드가 열리면, 화면의 좌측 플러스(“+”) 아이콘을 클릭합니다.

“서버만들기” 팝업이 뜨면 [직접만들기] 버튼을 클릭합니다.

이어서 [나와친구들을 위한 서버] 버튼을 클릭합니다.

서버 이름은 자유롭게 정하고 [만들기] 버튼을 클릭합니다.

서버가 만들어지면 채팅 채널란 밑에 “# 일반” 혹은 “# general” 의 설정 아이콘(채널 편집) 을 클릭합니다.

아나콘다 설치#

자동매매 코드 작성을 위해 필요한 환경들을 설치하는 시간을 갖겠습니다. 먼저 수백 개의 파이썬 패키지를 한 번에 사용할 수 있는 아나콘다 Anaconda 를 설치하겠습니다. 아나콘다 홈페이지(https://www.anaconda.com/products/distribution) 하단에서 [Download] 를 클릭합니다.

32-Bit 버전을 설치합니다.

다운로드 받은 아나콘다 설치 파일을 마우스 우클릭하여 관리자 권한으로 실행 합니다.

[Next] 를 클릭하여 아나콘다 설치를 시작합니다.

[I Agree] 를 클릭하여 라이선스 동의서에 동의합니다.

[All Users] 체크 후, [Next] 를 클릭하여 설치 타입을 정합니다.

[Next] 를 클릭하여 기본 설치 경로를 따릅니다.

[Register Anaconda3 as the system Python 3.9] 체크하여 Python IDE에서 인터프리터를 자동 인식 할 수 있도록 설정 합니다. 체크 후, [Install] 클릭하여 설치를 시작합니다.

PyCharm 설치#

파이참 PyCharm 은 파이썬 코드를 실행하고 그 결과를 확인하는 IDE 입니다. (IDE 설명 추가) PyCharm 파이참 홈페이지에서 (https://www.jetbrains.com/ko-kr/pycharm/download/#section=windows) Community 버전을 다운 받습니다.

다운로드 받은 설치 파일을 실행 시키고, Welcome Page 가 뜨면 [Next] 버튼을 클릭 합니다.

설치 경로를 설정한 후, [Next] 버튼을 클릭 합니다.

설치 옵션 화면에서 모든 옵션을 선택 후, [Next] 버튼을 클릭 합니다.

시작 메뉴 이름 설정 후, [Install] 버튼을 클릭 합니다.

마지막으로 재부팅을 바로 실행하거나 나중에 실행하기 선택 후 [Finish] 버튼을 클릭 합니다.
